Backbone is a web service for managing the Tuist project and making decisions. 
Among others it provides an API for collecting anonymous statistics about the usage of the tool.

It's built in Rails and hosted on Heroku.

## Setup

1. Git clone the repository: `git clone git@github.com:tuist/backbone.git`.
2. Install dependencies with `bundle install`.
3. Run the app with `bundle exec rails start`.
